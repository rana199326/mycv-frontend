# MyCV - Frontend

A modern CV Management web application built with React and TypeScript.

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Axios

## Features

- User authentication
- CV management
- Job offers
- Interview management
- Responsive design
