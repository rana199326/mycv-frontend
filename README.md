# MyCV - Frontend

A modern Full Stack CV Management application.

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- JWT Authentication

## Features

- User authentication
- Profile management
- CV management
- Skills management
- Languages management
- Education management
- Experience management
- Employer management
- Interview management

## Installation

```bash
npm install
npm run dev
```

## Backend

This frontend communicates with the MyCV Backend API built with NestJS and Prisma.
